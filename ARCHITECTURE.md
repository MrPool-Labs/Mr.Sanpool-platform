# Architecture

The MrPool ecosystem is composed of three layers:

mrpool-core → modeling  
mrpool-enterprise → decisioning  
mrsanpool-platform → control plane  


## Layer interaction

- mrpool-core produces structured signals  
- mrpool-enterprise produces decisions  
- mrsanpool-platform orchestrates control and governance  


## Design approach

- separation of concerns  
- modular architecture  
- control-oriented design  


## Note

This document provides a high-level overview only.
